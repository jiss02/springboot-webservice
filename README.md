# Spring Boot Practice 
스프링 부트로 test code를 작성해보고 CRUD, Login을 구현해보자!

## Requirements
For building and running the application you need:
- JDK 1.8
- Maven 3

## API
| URL                                         | Method |
| ------------------------------------------- | ------ |
| `/api/v1/posts`                     | GET    |
| `/api/v1/posts` | POST    |
| `/api/v1/posts/{id}`           | GET    |
| `/api/v1/posts/{id}`     | PUT    |
| `/api/v1/posts/{id}`       | DELETE    |
